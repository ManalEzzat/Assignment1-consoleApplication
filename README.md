This project is l Console Application developed in C#. It simulates a real-world digital banking experience where users can manage their finances
through a secure and interactive interface. The program focuses on robust logic, data integrity, and user-friendly interaction.
Secure User Authentication: The system uses a multi-step login process requiring a unique Account ID and a matching PIN.
Dynamic Balance Inquiry: Users can view their real-time balance at any time during the session.
Fund Deposit: Allows users to add money to their accounts, with immediate updates to the system's data.
Smart Withdrawal: A logic-gated feature that verifies if the account has sufficient funds before allowing a transaction, preventing negative balances.
Inter-Account Transfer: Enables users to transfer funds to another existing account by validating the recipient's ID and ensuring the sender's balance is adequate.
